# 🏠 NoBroker Clone

A pixel-perfect clone of the NoBroker platform built with **Next.js**, replicating the original design, user experience, and core functionalities. This project demonstrates modern frontend development practices, responsive design implementation, reusable component architecture, and real-world application development.

## 🚀 Overview

This project recreates the NoBroker website with a strong focus on visual accuracy and user experience. Every major feature and interaction has been implemented to closely match the original platform, providing a seamless property search and browsing experience across all devices.

## ✨ Features

- 🏠 Buy, Rent, and Commercial Property Listings
- 🔍 Advanced Property Search & Filtering
- 📍 Location-Based Property Discovery
- ❤️ Save & Manage Favorite Properties
- 📄 Detailed Property Information Pages
- 🖼️ Property Image Galleries
- 📱 Fully Responsive Design

## ⚡ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/your-username/nobroker-clone.git
```

### Navigate to Project

```bash
cd nobroker-clone
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Open Browser

```bash
http://localhost:3000
```
